## (a) On-the-fly analytic targets on the truth-free tail

  holding `self.last_channels` = the raw [Ndot, Nddot, N3dot] prediction of the

- Removes the M_max=3-at-stride-3 wall: self-consistency supervision at any
  depth without new data.

## (b) Von Neumann penalty (REPLACES the annulus hinge)

Frozen-coefficient amplification of the CLOSED scheme per sample, per radial
shell kappa, using the sample's own sigma-hat (model's context estimator,
sigma_hat_spec — no extra FFTs, the stepper computes it anyway):

  implicit symbol  I(k) = L_hat(k) + coef_R * L_hat^3(k)          [R^-1 fold]
  explicit symbol  E(k) = i*sigma_hat(k)                          [frozen advection]
                        + coef_S * L_hat^2(k) * i*sigma_hat(k)    [L^2 N fold]
                        + coef  * (1/12) * (L_hat*T_Ndot(k) - 5*T_Nddot(k))
  T_c(k) = the LINEARIZED per-channel transfer of the learned piece: TimeFD
  weights (frozen W_unit/dt^k) x the MODULATED width-15 tap trig polynomials
  (delta_taps from TapModulator — differentiable in theta) x the chain-rule
  mix binomials, with every Jacobian product frozen to the i*sigma_hat(k)
  advection (the Wiener linearization, error_analysis_full.tex sec. 7).
  AB2CN2 companion per shell: with r = (1 - dT/2 I)^-1,
    a = r*(1 + dT/2 I),  b = (3/2) dT r E,  c = -(1/2) dT r E
    G_eff(k) = spectral radius of [[a+b, c],[1,0]]  (closed-form 2x2 eigs,
    complex, differentiable)
  L_stab = lambda * mean_k[ relu(|G_eff(k)| - (1-eps)) ]^2,  eps = 0.02
  lambda swept {0.1, 1.0, 10.0} — the ONLY new hyper; three run names
  rollout_ft_p1_lam{01,1,10}.
- Horizon-free certificate: per-step contraction in the frozen model — removes
  the 16-step asterisk structurally. CAVEAT (document + email): linearized,
  r2/r3 neglected; P0's dissipative projection stays as the unconditional
  inference backstop.
- Report figure: max_k |G_eff| histogram over val samples per epoch (the
  certificate made visible — paper figure).
