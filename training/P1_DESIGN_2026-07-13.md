## (b) Von Neumann penalty (REPLACES the annulus hinge)

Frozen-coefficient amplification of the CLOSED scheme per sample, per radial
shell kappa, using the sample's own sigma-hat (model's context estimator,
sigma_hat_spec — no extra FFTs, the stepper computes it anyway):

  implicit symbol  I(k) = L_hat(k) + coef_R * L_hat^3(k)          [R^-1 fold]
  explicit symbol  E(k) = i*sigma_hat(k)                          [frozen advection]
                        + coef_S * L_hat^2(k) * i*sigma_hat(k)    [L^2 N fold]
                        + coef  * (1/12) * (L_hat*T_Ndot(k) - 5*T_Nddot(k))
  T_c(k) = the LINEARIZED per-channel transfer of the learned piece: TimeFD
  weights (frozen W_unit/dt^k) x the MODULATED width-15 tap trig polynomials
  (delta_taps from TapModulator — differentiable in theta) x the chain-rule
  mix binomials, with every Jacobian product frozen to the i*sigma_hat(k)
  advection (the Wiener linearization, error_analysis_full.tex sec. 7).
  AB2CN2 companion per shell: with r = (1 - dT/2 I)^-1,
    a = r*(1 + dT/2 I),  b = (3/2) dT r E,  c = -(1/2) dT r E
    G_eff(k) = spectral radius of [[a+b, c],[1,0]]  (closed-form 2x2 eigs,
    complex, differentiable)
  L_stab = lambda * mean_k[ relu(|G_eff(k)| - (1-eps)) ]^2,  eps = 0.02
  lambda swept {0.1, 1.0, 10.0} — the ONLY new hyper; three run names
  rollout_ft_p1_lam{01,1,10}.
- Horizon-free certificate: per-step contraction in the frozen model — removes
  the 16-step asterisk structurally. CAVEAT (document + email): linearized,
  r2/r3 neglected; P0's dissipative projection stays as the unconditional
  inference backstop.
- Report figure: max_k |G_eff| histogram over val samples per epoch (the
  certificate made visible — paper figure).

## Transfer-assembly specifics (verified in model_cond_local.py source)

- delta_taps: TapModulator(feats)->(B, C=2*n_ord, 2 dirs, W=15), zero-init head;
  applied in _mod_grads as per-sample grouped 1D convs, circular pad, scaled
  1/dx (x) and 1/dy (y) — symbol of the x-modulation for channel c:
  What_c(theta_x) = (1/dx) * sum_j dtap[c,0,j] * exp(-i*(j-pad)*theta_x),
  theta = 2*pi*k_mode/N (pad = W//2 centering). Same for y. Amplitude scale:
  (dT/dT_ref)^(S-k) per channel via k_of_channel (k=0 channels amp=0).
- Base gradient stencils: parent SpatialGrad width-15 (dimensionless taps /dx
  at forward) — base symbol assembled the same way from self.grad weights.
- sigma-hat context at rollout: context_feats_from_spectral(qh0, qh0-qh_m1)
  — ZERO extra FFTs in the stepper path; the same sig array (pre-shell-select)
  is what G_eff needs per shell: reuse sigma_hat_spec output (B, n_sh) mapped
  to the kappa grid of shells.
- Frozen per-channel output transfer (the linearization): channel outputs
  [Ndot,Nddot,N3dot] = mix(binomial) over Jacobian features; freeze each
  J(a,b) product to i*sigma_hat(kappa)*(-ik contraction) — net per-order
  T_c(kappa) = [chain-rule binomial row c] . [products of (TimeFD row symbol)
  x (grad symbol base+delta) x (i sigma_hat)] — implement as a small
  assemble_Geff(model, sig_shells, dT, L_hat_shells, coefs) returning
  |G|(B, n_sh), torch-differentiable through delta_taps AND base taps.
- coef/coef_R/coef_S: read from the closure arm's assembly in
  rollout_aposteriori.run_arm (inference decomposition block; I4 K-fold
  convention with (1-1/K^2) — copy the exact expressions).
