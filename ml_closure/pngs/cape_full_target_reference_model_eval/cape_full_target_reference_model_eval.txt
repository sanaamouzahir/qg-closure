WHAT THIS FOLDER IS (2026-07-13, cape full-target reference eval)

S4 a-priori evaluation package of the CAPE FULL-TARGET REFERENCE MODEL:
the preserved best checkpoint (epoch 65) of the cape full-target
Gaussian-filter training piff_cape_gauss, which was KILLED on 2026-07-13
when Sanaa changed the target convention to Jacobian-only Pi. The
checkpoint is kept frozen as the full-target reference — every figure
here exists to be compared, panel for panel, against the Jacobian-only
cape run (piff_cape_gj) currently training.

MODEL / DATA
  Checkpoint  ml_closure/runs_piff/piff_cape_gauss/best.pt (epoch 65;
              best training-time val R2 0.8132)
  Target      FULL Pi_FF (Jacobian + body terms), gaussian filter variant
  Members     FPCape-const, FPCape-sine, FPCape-ramp, FPCape-ou, FPCape-tel
  Val set     220 frames, 40,813,740 valid pixels (sponge/body masked)
  Eval jobs   1832563 (eval_piff.py package), 1832564 (symlog replot),
              config conf_piff_cape_cond_gaussian.yaml

HEADLINE NUMBERS (global, pooled over all val pixels)
  r2 0.8288   rmse 86.2   nll 6.67
  mean predictive sigma 309.5 vs rmse 86.2  ->  sigma ~3.6x overdispersed;
  empirical coverage 0.9992 at 1 sigma (nominal 0.68) — the same sigma
  OVERDISPERSION flag raised on CP-ML-1 S4. Skill splits by inlet forcing:
  r2 0.66/0.70 in the two lowest-zeta bins vs 0.92-0.95 in the top four,
  sigma flat (~309.5) in every bin (full table in the yaml copy in
  yamls/cape_full_target_reference_model_eval/).
  ARD lengthscales: zeta 8.37, zeta_dot 5.92, grad 11.48.
